
# Feasto


## Features
- Responsive website


## Tech Stack
- React
- Tailwind CSS
- Vite
- Redux

## How to use
1. Clone the repo
``` bash
git clone https://github.com/Althaf27/FeastoWebsite.git
```

2. Install dependencies
``` bash
npm install
```

3. Start the dev server
``` bash
npm run dev        
```
